
# Cyrillus - Shopify Theme

## Requirements

- [Shopify CLI for themes](https://shopify.dev/themes/tools/cli)
- [Node.js LTS & npm](https://nodejs.org/en/)

## Getting Started

### Install dependencies

```bash
npm install
```

### Start development

```bash
npm run dev
```

### Run test console

```bash
npm test
```

### Compile **src** folder

```bash
npm run build
```

### Clean compiles files

```bash
npm run clean
```
